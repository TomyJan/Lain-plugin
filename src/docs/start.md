## 安装插件
克隆项目，在`Miao-Yunzai根目录`执行


::: tabs

@tab Github

```bash
git clone --depth=1 https://github.com/Zyy955/Lain-plugin ./plugins/Lain-plugin
```

@tab Gitee

```bash
git clone --depth=1 https://gitee.com/Zyy955/Lain-plugin ./plugins/Lain-plugin
```

:::

## 安装依赖

::: tabs

@tab pnpm

```bash
pnpm install
```

@tab npm 

```bash
npm install
```

@tab yarn

```bash
yarn install
```
:::
