                                          # 获取QQBot机器人

                                          温馨提示

                                          请勿向购买任何人官方Bot资格，人财两空分分钟。

                                          目前QQBot仅对企业用户开放，个人开发者暂不开放。

                                          QQ开放平台
                                          有光的地方就有希望
                                          QQ机器人文档
                                          正如影子随光而生，希望伴随着光明永存

                                          前往 QQ开放平台 -> 登录(企业) -> 应用管理 -> 创建机器人

                                          前往 应用管理 -> 机器人 -> 开发 -> 开发设置

                                          字段类型必须说明
                                          机器人QQ号string否
                                          AppID (机器人ID)string是
                                          Token(机器人令牌)string是机器人身份调用 openapi，后续废除
                                          AppSecret(机器人密钥)string是用于在 oauth 场景进行请求签名的密钥

                                          以上操作，均在QQ开放平台完成!

                                          # 配置QQBot机器人

                                          温馨提示

                                          如果你没有登录QQ，可以在CMD等命令控制台直接执行指令，像QQ一样！

                                          目前QQBot仅对企业用户开放，个人开发者暂不开放。

                                          # 基本指令

                                          全量
                                          #QQBot设置 沙盒:私域:机器人ID:机器人令牌:机器人密钥
                                          单独启用群Bot
                                          #QQ群设置 沙盒:私域:机器人ID:机器人令牌:机器人密钥
                                          单独启用频道Bot
                                          #QQ频道设置 沙盒:私域:机器人ID:机器人令牌
